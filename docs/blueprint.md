# **App Name**: Astro Twitch Clash

## Core Features:

- Chat Bingo: Generate randomized bingo cards with common Twitch chat phrases and highlight phrases as they are said during the stream.
- Custom Phrase Addition: Allow users to add their own custom phrases to the bingo cards.
- Chat Tag: Implement a Chat Tag game where users can tag other users seen in the chat, earning points for tagging and losing points for being tagged as 'it'.
- Community List and Avatars: List all registered users in the community with their avatars, indicating who is currently active in the app and playing.
- Discord Bot Integration: Integrate with Discord using the provided bot token, server ID, and Twitch token to list the community members. Secure storage of the discord token for chat announcements.
- Live Leaderboard: Create a real-time leaderboard that tracks user scores in both Chat Bingo and Chat Tag, allowing for embedding in Discord and viewing within the app.
- Persistent Game Data: Store game data and user profiles using a cloud-hosted database to maintain persistence across sessions.

## Style Guidelines:

- Primary color: Deep space blue (#000080) to evoke the vastness of the cosmos.
- Background color: Starry black (#000000) with subtle animated stars.
- Accent color: Neon purple (#DA70D6) to highlight interactive elements like nebulae.
- Headline font: 'Orbitron' (sans-serif) for a futuristic and space-themed look. Body text: 'Roboto' (sans-serif) for clear readability.
- Use cosmic icons representing planets, stars, galaxies, and spacecraft to enhance the visual appeal.
- Design a layout that resembles a spaceship dashboard, utilizing panels and displays to organize information effectively.
- Implement smooth transitions with stardust effects and engaging animations for highlighting bingo phrases and displaying score updates, resembling cosmic events.
