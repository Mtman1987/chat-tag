Sound asset drop points
=======================

Chat Tag overlay sounds should be placed in:

public/sounds/chat-tag/

Expected filenames:

- tag.mp3: normal tag card
- pass-used.mp3: pass used to tag someone
- pass-granted.mp3: pass awarded
- ffa.mp3: free-for-all starts
- new-it.mp3: new IT announcement
- leaderboard.mp3: rolling leaderboard card
- score.mp3: score card
- live.mp3: live-view card
- history.mp3: recent activity/history card
- message.mp3: generic overlay message

The overlay falls back to the existing generated chime when a mapped file is missing.
